Merits and Demerits : 

1. RAW OCR Text :

Merits : 

*If OCR correctly captures: brand + product + variant -> embedding similarity becomes extremely discriminative.

*Embeddings Handle Minor OCR Noise ; "noodls masal"

*Captures Variant-Level Differences   (strawberry milk , chcolate milk etc)

Demerits : 

* OCR ineffective due to low quality image r tilted image ; Incomplete OCR can cause

map product to same brand but different product  or same product but different brand

* Text is in some style font instead of normal font -> OCR fails



2. VISUAL attributes DESCRIPTION SENTENCE 

Merits : 

Robust to slight motion blur and less quality images where complete text is not readable 

Demerits : 

Will have lot of false positives when specific brand is not identified ; it may to some other brand which has similar description


3. Structured Labels : 

L1: Brand category
L2: Subcategory
L3: Brand + product
L4: Brand + product + extra details



Merits : 

search space willbe resduced and L3 based filtering can reduce False positives

No need of extra embedding model

Demerits :

Brand should be correctly identified other wise it can map to someother brand
